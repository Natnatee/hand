# 🖐️ Hand Gesture Mouse Control

โปรเจกต์ควบคุมเมาส์บนหน้าจอคอมพิวเตอร์ด้วยการขยับมือผ่านกล้อง Webcam โดยใช้เทคโนโลยี Computer Vision ในการตรวจจับข้อต่อของมือ (Hand Landmarks) เพื่อแปลงเป็นการเคลื่อนที่ของเคอร์เซอร์และการคลิก

## ✨ คุณสมบัติหลัก (Features)

- **Calibration Mode**: ระบบปรับจูนพื้นที่ใช้งาน (Interaction Zone) อัตโนมัติในตอนเริ่มต้น โดยการแบมือค้างไว้ 3 วินาที เพื่อให้เข้ากับระยะการขยับมือของผู้ใช้แต่ละคน
- **Smoothing Movement**: ระบบหน่วงการเคลื่อนที่ของเมาส์ให้นุ่มนวล ไม่สั่นไหว (Jitter-free) ช่วยให้ชี้ตำแหน่งที่ต้องการได้แม่นยำ
- **Intuitive Controls**:
    - **Move**: ใช้ปลายนิ้วชี้ในการเลื่อนเคอร์เซอร์
    - **Click**: นำนิ้วโป้งแตะข้อต่อโคนนิ้วชี้แล้วปล่อยทันที (**ภายใน 1 วินาที**) และถ้าคลิกซ้ำซ้อนกันในระยะเวลา 1 วินาที ระบบจะทำการ **Double Click** ให้โดยอัตโนมัติ
    - **Drag**: นำนิ้วโป้งแตะค้างไว้**เกิน 1 วินาที** ระบบจะล็อคเมาส์ลง (mouseDown) เพื่อให้สามารถลากหน้าต่างหรือวัตถุได้ (Dragging)
- **Visual Feedback**: หน้าต่างแสดงผลลัพธ์แบบ Real-time พร้อมสถานะการทำงาน (Searching, Moving, Pinching, Dragging)

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Python**: ภาษาหลักในการพัฒนา
- **OpenCV**: สำหรับเปิดใช้งานกล้องและประมวลผลวิดีโอ
- **MediaPipe**: ตัวตรวจจับจุดเชื่อมต่อของมือ (Hand Landmarks) แบบประสิทธิภาพสูง
- **PyAutoGUI**: สำหรับควบคุม Mouse Interaction ในระดับ OS
- **NumPy**: สำหรับการคำนวณและแปลงพิกัด (Coordinate Mapping)

## 🚀 การติดตั้งและเริ่มใช้งาน

1.  **ติดตั้งไลบรารีที่จำเป็น**:
    ```bash
    pip install opencv-python mediapipe pyautogui numpy
    ```

2.  **รันโปรแกรม**:
    ```bash
    python main.py
    ```

3.  **ขั้นตอนการใช้งาน**:
    - เมื่อเปิดโปรแกรม จะเข้าสู่ **Calibration Mode** ให้ผู้ใช้แบมือออกหน้ากล้องและค้างไว้ 3 วินาที (สังเกตกรอบสีเหลืองและตัวเลข Countdown)
    - หลังจาก Calibration เสร็จ โปรแกรมจะสร้างกรอบสีชมพู (Interaction Zone) ให้ผู้ใช้ขยับนิ้วชี้ภายในกรอบนี้เพื่อเลื่อนเมาส์
    - **กด 'c'**: เพื่อทำการ Calibrate ใหม่
    - **กด 'q'**: เพื่อปิดโปรแกรม
